# Creating Layered Geospatial Representations from Humanities Data in ArcGIS
Repository for the workshop "Creating Layered Geospatial Representations from Humanities Data in ArcGIS" to take place at the Reorienting the Holocaust training session at the University of Southampton on June 2, 2026.

## Pre-Workshop Requirements

Before the start of the workshop, ensure the following:
- [ ] Participants have created a public ArcGIS account using their personal email. On [this webpage](https://doc.arcgis.com/en/arcgis-online/get-started/create-account.htm), scroll down to "Create a public account using an ArcGIS login."
- [ ] Participants have access to the [Github repository](https://github.com/laurisasastoque/creating-layered-geospatial/).

This repository will hold a guide as well as data that we will be using during the workshop. No need to download before the day, just come ready to learn!
